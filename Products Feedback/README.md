# 🌟 Product Satisfaction Survey

A beautiful, interactive survey webpage with a stunning night sky theme, designed to collect feedback about "The Afaha's Family Product" - you! 😊

![Survey Preview](https://via.placeholder.com/800x400/6a5acd/ffffff?text=Beautiful+Survey+with+Night+Sky+Theme)

## ✨ Features

- **🎨 Beautiful Design**: Night sky theme with animated twinkling stars and shooting stars
- **📱 Responsive**: Works perfectly on desktop and mobile devices
- **🔒 Password Protected**: Secure admin panel to view responses
- **💾 Local Storage**: All responses saved in browser storage
- **📊 Export Data**: Download responses as CSV file
- **♾️ Multiple Submissions**: Users can take the survey multiple times
- **⭐ Interactive Rating**: 5-star rating system with smooth animations

## 🚀 Quick Start

### Option 1: Use Directly
1. Download the `survey.html` file
2. Open it in any web browser
3. Start collecting responses!

### Option 2: Deploy Online
1. Upload to GitHub Pages, Netlify, or any web hosting service
2. Share the link with friends
3. View responses using the password

## 📋 How to Use

### For Survey Takers:
1. Click "Start Survey"
2. Answer all questions (optional fields can be skipped)
3. Submit your responses
4. Option to take the survey again

### For Admin (You):
1. Click "View Responses"
2. Enter password: `Li_Sa`
3. View all submissions with timestamps
4. Export data to CSV or clear all responses

## 🛠️ Technical Details

- **Pure HTML, CSS, and JavaScript** - No frameworks or dependencies
- **Local Storage** - Data persists in browser
- **Glass Morphism UI** - Modern frosted glass design
- **CSS Animations** - Smooth transitions and effects
- **Mobile-First** - Responsive design
